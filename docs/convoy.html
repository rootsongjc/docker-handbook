convoy

Convoy

Convoy是Rancher公司的产品，目前已开源https://github.com/rancher/convoy

A Docker volume plugin, managing persistent container volumes.

后端存储支持三种方案：

Device Mapper Amazon Elastic Block Store Virtual File System/Network File System Device Mapper

Amazon Elastic Block Store

Virtual File System/Network File System

该项目Rancher官方已经不维护了。不建议使用，目前使用也有问题，创建的backup无法显示正确的URL地址。目前已知唯品会在使用convoy。
